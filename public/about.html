Tools

HTML: HTML was used in this web application to complete the general structure. HML elements serve as the building blocks of this application.

CSS: CSS was used to provide a uniform (and at times unique) presentation within this application. In accordance with HTML, CSS allows for more exciting and user-friendly webpages.

JavaScript: JavaScript allowed us to make these webpages interactive. We can run it within our application to add functionality, and ultimately make our pages more exciting and useful for the user.

Vue.js: Vue was used to build our UI. Element binding allowed us to make our application more dynamic.

jQuery: jQuery was used to simplify our HTML and CSS code.

Express: Express was used to help us build our web application.

SQLite3: SQLite3 is an embedded database software used for local/client data storage.
